Welcome to the Dentica Dental Clinic Website Template!

This template is designed to provide a comprehensive and user-friendly experience for dental clinics looking to establish an online presence. Below are the details and instructions for using this template effectively.

### Project Structure
The project is organized into two main directories: `Documentation` and `Main Files`.

- **Documentation**: Contains instructions on how to install, edit, and customize the template.
- **Main Files**: Contains all the essential HTML files for the website, along with assets such as CSS, JavaScript, images, and fonts.

### Getting Started
1. **Installation**: 
   - Download the template files and extract them to your desired location.
   - Open the `Main Files/index.html` in your web browser to view the homepage.

2. **Customization**:
   - Modify the HTML files in the `Main Files` directory to suit your clinic's needs.
   - Update the CSS files in `Main Files/assets/css/style.css` for custom styling.
   - Add or replace images in the `Main Files/assets/img` directory.

3. **JavaScript Functionality**:
   - The `Main Files/assets/js/main.js` file contains scripts for interactivity. Customize this file as needed.

### Pages Included

### Assets
- **CSS**: Bootstrap and custom styles are included in the `assets/css` directory.
- **JavaScript**: Bootstrap and custom scripts are included in the `assets/js` directory.
- **Images**: Organized into subdirectories for easy access.
