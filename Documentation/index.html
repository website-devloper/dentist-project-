Dentica Template Documentation

Installation

To install the Dentica template, simply download the zip file and extract it to your desired location. Open the index.html file in your web browser to view the homepage.

Editing

You can edit the HTML files located in the Main Files directory to customize the content of your website. Use any text editor or IDE of your choice.

Customization

For styling, you can modify the style.css file located in Documentation/assets/css or Main Files/assets/css. You can also add your own CSS files in these directories.

Images

Place any images you want to use in the Documentation/assets/img directory. Ensure to reference them correctly in your HTML files.
