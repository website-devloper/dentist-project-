Installation

To install the Dentica template, simply download the zip file and extract it to your desired location. Open the Main Files/index.html file in your web browser to view the homepage.

Editing

You can edit the HTML files located in the Main Files directory to customize the content of your website. Use any text editor or IDE of your choice. The main files to edit are:

  • index.html - The main homepage.
  • book-appointment.html - The appointment booking page.

Customization

For styling, you can modify the style.css file located in Main Files/assets/css. You can also add your own CSS files in this directory. The main CSS file is style.css, and the responsive styles are in responsive.css.

Images

Place any images you want to use in the Main Files/assets/img directory. The images are organized into subdirectories based on their purpose (e.g., doctors, hero, logos, services, testimonials). Ensure to reference them correctly in your HTML files.

404 Page

A `404.html` page has been included in the `Documentation` folder. You can customize this page and move it to the root of your website to handle "page not found" errors.
