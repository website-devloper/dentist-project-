HTML Structure

This theme uses a standard HTML5 structure with semantic tags. The layout is based on the Bootstrap 5 grid system.

<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Meta Tags -->
    <!-- CSS Links -->
</head>
<body>
    <!-- Header -->
    <header>...</header>

    <!-- Main Content -->
    <main>...</main>

    <!-- Footer -->
    <footer>...</footer>

    <!-- JS Scripts -->
</body>
</html>

CSS Files

We use standard CSS files located in assets/css.

  • bootstrap.min.css - Bootstrap 5 framework
  • style.css - Main theme styles
  • responsive.css - Responsive adjustments (if applicable, or included in style.css)

JavaScript Files

JavaScript files are located in assets/js.

  • jquery.min.js - jQuery library
  • bootstrap.bundle.min.js - Bootstrap JS
  • main.js - Theme custom scripts

Customization

Changing Colors

You can change the main colors by editing the CSS variables in assets/css/style.css:

:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
}
