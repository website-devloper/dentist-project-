# Dentica Dental Clinic Website Template

Welcome to the Dentica Dental Clinic Website Template! This template is designed to provide a clean and modern interface for dental clinics looking to establish an online presence. Below you will find information on how to set up, customize, and use this template.

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Customization](#customization)
- [File Structure](#file-structure)
- [Credits](#credits)
- [License](#license)

## Features

- Responsive design using Bootstrap
- Multiple pages including Home, About, Services, and Contact
- Appointment booking form
- Testimonials section
- Easy customization options

## Installation

1. **Download the Template**: Obtain the `dentica.zip` file from the source where you purchased or downloaded it.
2. **Extract the Files**: Unzip the `dentica.zip` file to your desired location on your computer.
3. **Open the Project**: Use a web browser to open the `index.html` file located in the `Main Files` directory to view the homepage.

## Customization

- **CSS Customization**: Modify the `style.css` file located in `Main Files/assets/css/` to change the appearance of the website.
- **JavaScript Customization**: Add or modify functionality in the `main.js` file located in `Main Files/assets/js/`.
- **Images**: Replace images in the `Main Files/assets/img/` directory with your own images to personalize the site.
